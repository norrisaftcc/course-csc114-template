# The Plan

CSC 114 will be piloted as an 8 week Summer session. 

Basic plan:

0: Orientation / Fundamental Skills Refresh
1: Building Cloud Agents (example: Claude Console)
2: Begins Manning material
3+: Continues Manning material
8: Project module

Obviously in 8 weeks we're not covering all of the textbook "Deep Learning with Python" 3e, Chollet & Watson.
However, we're definitely introducing the basics of Keras, PyTorch, Tensorflow, and then building some of the topics such as MNIST. Later, advanced portions of the book, which might later be in a future semester, can also be used by individual students as a project.

# Notes on Pedagogy

We will use the workflow:

Learn -> Practice -> Apply -> Assess

Learn involves activities and material that introduce vocabulary, topics, and workflows.
Practice involves "retry until successful" grading (simply making sure you're ready to continue)
Apply involves hands on work and more detail applied.
Assess involves Homework assignments, where the user is being assessed on material they have completed the previous steps on.

Labs: Practice or Apply
Homework: Assess

Bloom's Taxonomy can be tackled in any order in this class: I sometimes say that it now starts with "Create", which used to be the last step. (The difference of course; no matter where you start, you still need to do the other steps.)
